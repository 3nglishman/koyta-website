# Koyta Hope & Rift Initiative Website

Official website for Koyta Hope & Rift Initiative, a Canadian nonprofit dedicated to restoring dignity in Kipsigis communities through community-owned programs in Kenya's Rift Valley region.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **CMS**: Directus (https://cms.koyta.org)
- **Hosting**: Vercel
- **Assets**: Cloudflare R2

## Getting Started

1. Clone the repository
2. Copy `.env.example` to `.env.local`
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000)

## Deployment

This site auto-deploys to Vercel when pushing to the `main` branch.

## Content Management

Content is managed through Directus CMS at https://cms.koyta.org
